This android/java app is a close replica of the game 98 cards which you can find on Youtube at https://www.youtube.com/watch?v=KzLhZz3XHt0 and on the android app store.

Some gradle properties files included, you might want to rebuild from source files, which are found in \app\src\main\java\com\example\tpfinal

Done as a school project



Rules of the game:

Place higher cards on ascending piles.
Place lower cards on descending piles.

You can place a card that is exactly 10 lower on an ascending pile.
You can place a card that is exactly 10 higher on an descending pile.

Time between each placement influences score.
